# 🏗️ EPH-B Design & Construction Website

## 📌 Overview

This is a modern, responsive website for **EPH-B Design & Construction**, a Malawian-based company offering premium architectural design and construction services.

The website showcases the company’s services, portfolio, and contact information while providing a professional digital presence for client engagement.

---

## 🎯 Features

* ✅ Responsive modern UI design
* ✅ Interactive hero slider
* ✅ Company portfolio showcase
* ✅ Services section with detailed offerings
* ✅ Contact section with direct WhatsApp integration
* ✅ Social media integration
* ✅ Clean navigation and user-friendly layout

---

## 🏢 About the Company

**EPH-B Design & Construction** specializes in:

* Architectural Design
* Full Construction Services
* Commercial Projects
* Surveying Services

The company focuses on delivering **high-quality, innovative, and sustainable building solutions** in Malawi.

## 🛠️ Technologies Used

* **HTML5** – Structure of the website

* **JavaScript** – Interactivity (slider, navigation, etc.)

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/ephb-website.git
   ```

2. Navigate into the project folder:

   ```bash
   cd ephb-website
   ```

3. Open `index.html` in your browser.
